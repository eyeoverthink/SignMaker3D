## Font Optimization

### **Scott Algorithm Technology**

SignCraft 3D uses the **Scott Algorithm** for geometry optimization:

**Traditional Method:**
- 1000+ vertices per letter
- Large file sizes
- Slow 3D printing

**Scott Algorithm:**
- 33-163 vertices per letter
- **80-95% reduction**
- Fast exports

### **How It Works**

1. **Boundary Tracing** - Moore-Neighbor algorithm finds letter edges
2. **Path Simplification** - Douglas-Peucker reduces vertices
3. **Quality Preservation** - Maintains visual appearance

**Result:** Smaller files, faster prints, same quality.

## 3D Printing Guide

### **Recommended Settings**

**Material:** PLA or PETG  
**Layer Height:** 0.2mm  
**Infill:** 15-20%  
**Supports:** Usually not needed  
**Print Time:** 2-8 hours (depending on size)

### **Print Orientation**
- Lay flat for best quality
- Tubes print horizontally
- Backing plates print face-down

### **Post-Processing**
1. Remove supports (if any)
2. Sand rough edges (optional)
3. Paint or leave natural
4. Install LED strips

## LED Installation

### **LED Types Supported**

1. **5V Neon LED Strips** (Recommended)
   - Flexible silicone
   - Easy installation
   - Plug-and-play

2. **WS2812B Addressable**
   - Programmable colors
   - Requires microcontroller
   - Advanced effects

3. **Standard LED Strips**
   - Simple, cheap
   - Single color
   - USB powered

### **Installation Steps**

1. **Measure LED strip** to match tube length
2. **Insert into tube** (use split-tube design for easy access)
3. **Connect power** (USB 5V or battery pack)
4. **Test brightness** before final assembly
5. **Mount to backing plate**

### **Power Requirements**

- 5V USB: Up to 2A (10W)
- Battery pack: 4× AA (6V)
- LED consumption: ~60mA per meter

## Troubleshooting

### **Export Issues**

**Problem:** STL file won't open  
**Solution:** Check file size, re-export with lower quality

**Problem:** Missing geometry  
**Solution:** Increase wall thickness, check for thin sections

**Problem:** File too large  
**Solution:** Scott Algorithm should auto-optimize, check settings

### **Print Issues**

**Problem:** Tubes breaking  
**Solution:** Increase wall thickness to 3mm+

**Problem:** Poor surface quality  
**Solution:** Reduce print speed, increase layer height

**Problem:** Warping  
**Solution:** Use heated bed, add brim

### **LED Issues**

**Problem:** LEDs don't fit  
**Solution:** Use split-tube design, increase inner diameter

**Problem:** Uneven brightness  
**Solution:** Check LED strip quality, add diffuser

**Problem:** Power issues  
**Solution:** Check voltage (5V), use adequate power supply

## Tips & Tricks

### **Design Tips**

1. **Keep it simple** - Fewer details = easier printing
2. **Test small first** - Print a small version before full size
3. **Consider LED placement** - Design with LED strips in mind
4. **Use split tubes** - Makes assembly much easier

### **Optimization Tips**

1. **Let Scott Algorithm work** - Don't override simplification
2. **Use recommended fonts** - Some fonts optimize better
3. **Avoid tiny details** - Sub-2mm features may not print well
4. **Check preview** - Always preview before exporting

### **Production Tips**

1. **Batch printing** - Print multiple signs together
2. **Material efficiency** - Nest parts to save filament
3. **Quality vs speed** - Balance based on use case
4. **Post-processing** - Sand and paint for professional finish
